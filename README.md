# 🏥 Healthcare Waitlist Analysis Dashboard

## 📌 Project Overview
The Healthcare Waitlist Analysis Dashboard is an interactive Power BI project designed to monitor and analyze patient waiting lists across healthcare facilities. The dashboard provides valuable insights into waiting times, patient distribution, case types, and historical trends, helping healthcare stakeholders make informed, data-driven decisions.

## 🎯 Objectives
Monitor the latest healthcare waiting list.
Analyze waiting time across different case types.
Track historical changes in patient waitlists.
Compare waiting lists by age group and time band.
Provide interactive filtering for detailed analysis.

## 🛠️ Tools & Technologies
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel

## 📊 Dashboard Highlights
Latest Month Waitlist: 679K patients
Total Cases Analyzed: 25 Million
Median Wait Time: 6.9 Months

## 📈 Key Insights
The dashboard analyzed approximately 25 million healthcare cases, with 679,000 patients remaining on the latest monthly waiting list.
The median waiting time of 6.9 months indicates that many patients experience extended delays before receiving healthcare services.
Outpatient services recorded the highest average waiting time (7.5 months), followed by Day Case (5.4 months) and Inpatient (4.2 months), highlighting greater pressure on outpatient care.
Historical analysis shows a steady increase in total waiting lists over the reporting period, suggesting growing demand for healthcare services.
Interactive filters allow users to explore the data by Specialist, Case Type, Age Profile, Time Band, and Archive Date, making the dashboard suitable for detailed operational analysis.

## 💼 Business Value
This dashboard enables healthcare administrators and decision-makers to monitor service performance, identify high-demand areas, evaluate waiting time patterns, and support strategic resource planning through interactive data visualization.

## 📌 Conclusion
This project demonstrates the use of Power BI to transform large healthcare datasets into meaningful business insights. Through interactive dashboards and performance metrics, the solution supports effective monitoring of healthcare waiting lists and promotes data-driven decision-making for improved service delivery.
